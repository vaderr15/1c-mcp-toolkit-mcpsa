# 1С Запросы - Компактное руководство для Евлантия

## КРИТИЧНО
- НИКОГДА не угадывайте имена объектов/полей - используйте get_metadata
- Запросы ТОЛЬКО в одну строку
- Параметры через &Имя, не конкатенация строк

## Структура
```
ВЫБРАТЬ [ПЕРВЫЕ N] поля ИЗ таблица [ГДЕ условие] [УПОРЯДОЧИТЬ ПО поля ВОЗР/УБЫВ]
```

## Таблицы
- Справочник.Имя, Документ.Имя, РегистрНакопления.Имя, РегистрСведений.Имя
- Виртуальные: .Остатки(,фильтры), .Обороты(&Нач,&Кон,,фильтры), .СрезПоследних(&Дата,фильтры)

## Операторы
И,ИЛИ,НЕ,=,<>,МЕЖДУ,В(),ПОДОБНО "%шаблон%",ЕСТЬ NULL,В ИЕРАРХИИ(&Родитель)

## Паттерны
- Исключить удаленные: `ГДЕ НЕ ПометкаУдаления`
- Лимит: `ВЫБРАТЬ ПЕРВЫЕ 10`
- Ссылки: `{"_objectRef":true,"УникальныйИдентификатор":"uuid","ТипОбъекта":"тип"}`

## Workflow
1. get_metadata (если не знаете точные имена)
2. execute_query с параметрами
3. Готово!
